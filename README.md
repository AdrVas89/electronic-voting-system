# Electronic Voting System

This is a web-based Electronic Voting System developed for the Frontend Programming project. The system allows voters to register, log in, view active elections, cast one vote per election, and view results. Administrators can create elections, add candidates or voting options, and view election results.

## Technologies Used

- HTML
- CSS
- Bootstrap
- PHP
- MySQL
- XAMPP
- phpMyAdmin

## Main Features

- User registration and login
- Admin and voter roles
- Admin dashboard
- Election creation
- Candidate creation
- Voter dashboard
- Vote casting
- Duplicate vote prevention
- Automatic vote counting
- Results page
- Secure logout
